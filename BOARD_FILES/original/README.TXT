Bienvenue sur Hepialight

Vous pouvez demarrer directement a programmer en Python dans le fichier 'main.py'.

Pour un terminal serie:
 - Windows 10: clique sur l'icone 'Windows' puis recherche le mot 'périphérique'.
   execute alors le programme du même nom.
   la liste de tout les périphériques aparait dont 'Périphérique série USB(COM_)'
   note le numéro COM_.
   Télécharge le programe putty : https://the.earth.li/~sgtatham/putty/latest/w32/putty.exe
   Ouvre putty.exe, Sélectione 'Serial' dans l'option -Connection type- et change le numéro COM
   Tu peux maintenant cliquer sur Open.
 - Mac OS X: utilisez la commande: screen /dev/tty.usbmodem*
 - Linux: utilisez la commande: screen /dev/ttyACM0

Pour plus d'aide, allez sur le site www.hepialight.ch/aide/.
